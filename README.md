# English Learning API

REST API built with Spring Boot for an English learning platform.

## Current Features

- User registration
- DTO pattern implementation
- Bean Validation
- Entity to DTO mapping (UserMapper)
- Clean layered architecture

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Lombok

## Architecture

controller  
service  
repository  
dto  
mapper  
model  

## Endpoints

POST /api/v1/users  
GET /api/v1/users
